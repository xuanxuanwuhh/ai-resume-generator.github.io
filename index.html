系统架构

Backend Service

FastAPI OCR API

从 .env 读取阿里云密钥，对外暴露 /api/transcript/parse。

05

OCR API 后端

现在仓库中新增了 flutter-api/ 目录，直接包含可运行的 FastAPI 后端。 这样 Flutter 页面就不再只是“预留接口”，而是有实际可联调、可部署的服务端实现。

核心接口

POST /api/transcript/parse
Content-Type: multipart/form-data
file=<PDF / docx / image>

返回候选课程、OCR 原文和友好错误信息，供 Flutter 页面勾选导入。

后端技术

  • FastAPI + Uvicorn
  • PyMuPDF 处理 PDF
  • Pillow 压缩和校正图片
  • Aliyun OCR 统一识别接口

部署原则

  • 密钥只放后端 .env
  • 在线 Flutter 页面必须连接 HTTPS 后端
  • 前端只保存 API Base URL，不保存 AccessKey
